how to visit
1. open the Terminal application
  • on a mac, Terminal can be found under Applications->Utilities, or on windows you can use the Command Prompt
2. copy and paste the SSH command below into the Terminal then press enter 
ssh swim@0.tcp.ngrok.io -p1321
3. enter welcome as the password
4. roam our paths, visit our gallery and chat with strangers. here is a google doc with some tips on how to navigate around: oasis guide
oasis vr commune is a project by Max Fowler for the SFPC summer 2016 student showcase.
